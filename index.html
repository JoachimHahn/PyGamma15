About

We are holding a week-long (November 16 - 20, 2015) workshop on Python for gamma-ray astronomy at MPIK Heidelberg.

Why focus on Python?

Python has become the most popular programming language in astronomy (see "Software Use in Astronomy: an Informal Survey").
This is also the case in gamma-ray astronomy, where most open-source analysis codes are Python-based or have a Python interface (e.g. Gammapy, Naima, Gammalib, ctools, 3ML, FermiPy, gammatools, VHEObserverTools, ctapipe, Sherpa ).
Python can serve as the glue that brings together data from various gamma-ray telescopes (e.g. Fermi, H.E.S.S., VERITAS, MAGIC, HAWC, CTA ) as well as astrophysical modeling codes.

The goal is to grow the community of of developers of the various Python codes and to improve the capabilities and interoperability of the available codes.
Users that want to start using the tools for science analysis, or are interested to start collaborating on multi-mission analysis are also very welcome!

Sessions

We will start the workshow with plenary talks on Monday morning. Tuesday to Thursday is parallel tutorial, coding sprint and discussion sessions. On Friday we'll have a plenary lightning talk and summary session.

The rest of the week (Tuesday to Friday) will be largely a free-form hands-on work meeting. We expect participants to form small groups to discuss and work on what they like. To make this effective, we will have a board where everyone meets in the morning and after lunch (for ~ 15 minutes) to propose sessions that every participant can join. Every evening we'll again all meet to have a 5-minute summary of each session.

For users and new people that want to become contributors, we'll offer a series of tutorials: For further information, see the tutorials folder in the workshop repo on Github.

Prepare

If you're new to Python, we recommend that you spend some time to learn it before the workshop in the context of scientific computing and Astronomy.

  • The Python Scientific Lecture Notes are a good tutorial to learn about Numpy, Scipy and Matplotlib.
  • The Practical Python for Astronomers tutorials are a good resource to learn about Astropy and specifically the Fitting and Modeling 1-D and 2-D Data tutorial to get started with Sherpa.
  • Since Python is very popular overall, and specifically in Astronomy, there's of course a ton of other free tutorials, books and resources available online, e.g. the Python for Astronomers web site or the Astropy tutorials.
  • If you don't have a scientific Python stack installed on your computer, try it out online at https://try.jupyter.org/ (this uses the Jupyter notbook, formerly known as the IPython notebook) and then to install it on your machine, we recommend Anaconda as a free, cross-platform (Linux, Mac, Windows) scientific Python distribution.
  • If you are completely new to Python, and have time, you can of course read a Python programming book (there's some free books listed here). But it turns out that for effective scientific computing and data analysis, this isn't really needed, and some basic knowledge on functions, classes and modules can get you a long way!
  • Gamma-ray astronomy specific tutorials will be prepared for the workshop, although of course you can start learning about the existing codes using their normal documentation.

If you're thinking about contributing to one of the open-source projects discussed at this workshop, you should learn a bit about the git version control system and code development on Github. The Github bootcamp help page contains links to excellent tutorials and resources.
